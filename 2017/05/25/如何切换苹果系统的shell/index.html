如何切换 mac OS系统中的shell

列出系统已经安装的shell

cat /etc/shells
输出结果：

至于网上提到的另一种方法“ chsh -l ” ，在Mac 系统中亲测不能使用。

切换shell

chsh -s /bin/zsh

chsh–> “change shell”
